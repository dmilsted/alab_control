Welcome to EM Autoprep

This system provides automated control for electron microscope sample preparation. The available modules are:

Machine Test

Verify system configurations and test basic machine functions.

SEM Tray Preparation

Automated preparation of samples for Scanning Electron Microscopy, including voltage control and positioning.

SEM Stage Control

Precise control of the SEM stage position and movement.

TEM Preparation

Sample preparation controls for Transmission Electron Microscopy.

Using the System

  • Select the desired module from the navigation bar above
  • Use the terminal to monitor system responses and status
  • Follow the on-screen instructions for each module
  • Monitor the terminal output for feedback and error messages
